# 催促くん

> 「お金の催促」という心理的ストレスの高い行為を，ツールのせいにして笑いに変える，超軽量Webツール．

## 概要

友人・知人・同僚に対して「お金返して」と直接言いにくい場面で，**催促くんが代わりに角の立たない催促文を自動生成**します．

生成されたテンプレートをコピーしてLINEやメールに貼り付けるだけ．末尾の「催促くんにより自動生成されています」の一文が，受け取った側の怒りを笑いに変える安全弁として機能します．

## 機能

### 💰 お金の催促
- 複数人への一括催促（グループLINE対応）
- 割り勘（均等分割） / 個別金額の切り替え
- 3種テンプレート：丁寧系 / 事務的系 / 圧力系（督促状風）

### 📋 出欠確認
- イベント名・日時・場所・参加者リストを入力
- 3種テンプレート：カジュアル系 / 事務的系 / 圧力系（強制召喚風）
- `◯ / ✕` 回答欄付きテンプレート

### 共通機能
- 📋 ワンタップコピー（クリップボードAPI + フォールバック）
- 📱 スマホファースト設計
- 🔒 入力内容はブラウザ内で完結（サーバー送信なし）

## 使い方

ES Modules を使用しているため，ローカルサーバー経由で開いてください．

```bash
# Python を使う場合
python -m http.server 8000

# npx を使う場合
npx serve .
```

ブラウザで `http://localhost:8000` を開きます．

## 技術スタック

- **HTML / CSS / JavaScript**（フレームワーク不使用）
- **Google Fonts** — DotGothic16（見出し・ラベル）/ Zen Maru Gothic（本文）
- バックエンド・データベース・外部API：**不使用**
- すべての処理はクライアントサイドで完結

## デザインコンセプト

> RPGゲームのUI風パネルデザインをクリーンな配色で実現．

- ドットフォント（DotGothic16）でゲームUI感を演出
- 白・グレーベースのクリーンな配色（`#f0f0f0` / `#ffffff` / `#4a4a4a`）
- 二重ボーダーのRPGウィンドウ風パネル
- スマートフォンファースト設計
