# 催促くん

> 「お金の催促」という心理的ストレスの高い行為を，ツールのせいにして笑いに変える，超軽量Webツール．

## 概要

友人・知人・同僚に対して「お金返して」と直接言いにくい場面で，**催促くんが代わりに角の立たない催促文を自動生成**します．

生成されたテンプレートをコピーしてLINEやメールに貼り付けるだけ．末尾の「催促くんにより自動生成されています」の一文が，受け取った側の怒りを笑いに変える安全弁として機能します．

## 機能

- 📝 **入力フォーム** — 相手の名前・件名・金額・支払期日の4項目を入力
- 📄 **3種テンプレート生成** — 丁寧系 / 事務的系 / 圧力系（督促状風）
- 📋 **ワンタップコピー** — 生成テキストをクリップボードにコピー
- 📱 **スマホファースト** — モバイルブラウザに最適化

## デモ

```
支払督促通知書

受取人　田中 殿

上記の者に対し，飲み会代として立替済みの
金¥3,000也を3月15日までに支払うよう通知する．

※本通知は催促くん（https://...）により自動生成されています
```

## 使い方

### ローカルで実行

ES Modules を使用しているため，ローカルサーバー経由で開いてください．

```bash
# npx を使う場合
npx serve .

# Python を使う場合
python -m http.server 8000
```

ブラウザで `http://localhost:8000`　を開きます．

## 技術スタック

- **HTML / CSS / JavaScript**（フレームワーク不使用）
- **Google Fonts** — Noto Serif JP（見出し）/ Noto Sans JP（本文）
- バックエンド・データベース・外部API：**不使用**
- すべての処理はクライアントサイドで完結

## デザインコンセプト

> 「市役所のHP」「古い銀行サイト」レベルのお堅いデザインを意図的に採用．ダサさは狙い通りであり，バグではない．

- 明朝体（Noto Serif JP）を見出しに使用してお役所感を演出
- ダークレッド（`#8b0000`）をアクセントカラーに
- 枠線・ボーダーを多用した「書類感」のあるレイアウト
